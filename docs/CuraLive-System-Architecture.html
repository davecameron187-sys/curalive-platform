Entry Points — How Users Access the Platform

Web Browser

Operator, Admin, and Attendee access via browser. Full dashboard, OCC, event rooms.

Phone Dial-In

Attendees call the conference bridge with their unique PIN. OCC admits them.

External Bridge

3rd-party conference systems. Shadow Mode joins silently via Recall.ai.

CRM / Partner API

REST + Webhook integration. API key auth (clv_ prefix). Bulk registration.

Mobile Attendee

5-panel swipeable event room: Video, Transcript, Q&A, Polls, Info.

OAuth Login

Google / GitHub SSO. JWT session cookies. Role-based access control.

Key Data Flows — How Information Moves Through the Platform

1. Event Lifecycle

Booking Form → Event Created → Mailing List Import → Auto-PIN Generation → Invitation Emails Sent → Attendee Clicks → Registered → OCC Loads Event → Attendees Dial In / Join Web → Live Conference Runs → Post-Event Report Generated → Social Content Published

2. AI Intelligence Pipeline

Live Audio → Real-time Transcription → Sentiment Analysis (per speaker) → Q&A Auto-Triage → Speaking Pace Coaching → Live Rolling Summary → Tagged Metrics Extraction → Post-Event Brief + Recap → Social Media Content → Podcast Conversion

3. Shadow Mode Flow

External Bridge Configured → Recall.ai Bot Joins Call Silently → Audio Captured in Real-time → AI Transcription Runs → Sentiment + Intelligence Analysis → Feeds to OCC Dashboard → Operator Sees Live Insights → Zero Disruption to Original Call

4. Compliance Engine

Engine Runs Every 5 Minutes → Scans Registrations (fraud patterns) → Scans Audit Log (access anomalies) → Scans Exports (data exfiltration) → AI Assesses Severity (GPT-4o) → Threats Persisted → ISO 27001 + SOC 2 Scored → Predictive Alerts Generated

5. CRM / Partner API

Partner Gets API Key (clv_ prefix) → REST: Create Registrations → Bulk Import (up to 500) → Query Event Stats → Webhook on Registration → Zero-Click Pre-Register All → Confirmation Emails Sent → Full Audit Trail Logged

6. Health Monitoring

Health Guardian (30s cycle) → Checks: DB, Twilio, OpenAI, Ably, Recall, Active Events → Anomaly Detection (2.5σ threshold) → Auto-Incident Creation → AI Root-Cause Analysis → Customer Impact Reports → Cross-feeds Compliance Engine
