Testing Checklist — Shadow Mode

Before the event

  • Platform is open at the base URL above
  • Shadow Mode page loads at /shadow-mode
  • Meeting host briefed — admit "CuraLive Intelligence"
  • Valid Zoom / Teams join link in hand

After the session

  • Session status shows "Completed"
  • Transcript segments count is above 0
  • Tagged Metrics Dashboard shows 4 new records
  • Sentiment score populated (or noted as short event)
Quick Troubleshooting
Bot does not join

Check the meeting link is correct and the meeting is actually running. If there is a waiting room, the host must admit the bot.

Session shows Failed

Meeting link is wrong or meeting has not started yet. Create a new session with the correct link.

No transcript captured

Bot was not admitted to the meeting. For Zoom accounts with strict security, a Zoom admin must allow third-party bots.
