What CuraLive Is
CuraLive is a world-class AI-powered communication intelligence platform for regulated corporate communication events — earnings calls, AGMs, investor days, analyst briefings, regulatory disclosure sessions.
It is built around a filed patent specification covering 9 invention families including:

Cross-platform bridge capture via DTMF
Non-disruptive AI monitoring
Canonical signal normalisation
Multi-dimensional communication analysis — 7 concurrent AI pipelines
Embedded real-time operator intelligence interface
Cross-event benchmarking
Autonomous compliance intervention
Predictive risk and answer-risk architecture
Organisational memory graph
Autonomous policy synthesis
AEOS enterprise workflow state machine
Deterministic governance gateway
Identity-confidence fusion module

The platform has two distinct user experiences:
Shadow Mode — the operator platform. Used by CuraLive operators to run regulated events on behalf of client organisations. Dark theme. Operational precision.
CuraLive Insight — the client portal. Used by CCOs, IR Directors, and Chief Compliance Officers at listed companies. Light theme. Strategic intelligence. Inspiring daily experience.
