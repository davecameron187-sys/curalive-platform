Architecture Overview
WebPhone-First Flow
Plain Text
Session Creation
    ↓
Initialize WebPhone (Telnyx)
    ↓
Success? → Use WebPhone
    ↓
Failure → Retry (exponential backoff)
    ↓
Max retries exceeded? → Switch to fallback provider
    ↓
Notify operator of provider change
    ↓
Session continues with fallback provider
Shadow Mode Flow
Plain Text
Operator opens Shadow Mode
    ↓
Browse archived sessions (paginated, searchable)
    ↓
Select session → View details
    ↓
Run AI services (Whisper, Recall)
    ↓
Export as CSV/JSON
    ↓
Download or share
