SECTION A — Complete AI Services Inventory
A1. List every file in server/services/ with its file size so we can identify dead stubs vs real implementations:
bashls -la server/services/*.ts | awk '{print $5, $9}' | sort -rn
A2. For each service, we need to know if it's real code or a stub. Run:
bashfor f in server/services/*.ts; do lines=$(wc -l < "$f"); echo "$lines $f"; done | sort -rn
A3. For every AI service file, show the first 5 lines — enough to see if it has real logic or is empty/placeholder:
bashfor f in server/services/*.ts; do echo "=== $f ==="; head -5 "$f"; echo ""; done
A4. Which services are actually imported and used somewhere — versus which are orphaned dead code:
bashfor f in server/services/*.ts; do
  name=$(basename "$f" .ts)
  count=$(grep -r "$name" server/ --include="*.ts" -l 2>/dev/null | grep -v "$f" | wc -l)
  echo "$count imports: $name"
done | sort -rn
