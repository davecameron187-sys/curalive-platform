🎚️ CuraLive Operator Dashboard

Unified Control Center for Event Management & Operations

Design Philosophy: Operator-centric, visually stunning interface consolidating all daily tasks (OCC, webcasting, video, shadow mode, event management, participant tracking) into one intuitive workspace. Designed for professional broadcast control room operations.

🎚️ Unified Live Console

The heart of event operations. All critical controls and real-time monitoring consolidated into a single, operator-centric interface.

Key Features:

Event Control Panel (Left): Live status, participant count, quick actions (Start/Stop/Emergency Stop), event details, next event preview
Live Monitoring (Top Center): Real-time participant list with sentiment indicators, Q&A queue with pending questions
Operator Console - OCC (Middle): Live transcription display, sentiment analysis gauge (positive/neutral/negative), audio/video controls
Webcasting & Video (Bottom): RTMP stream status, bitrate, viewer count, video preview, recording status
Shadow Mode (Right): Toggle for monitoring-only mode, compliance alerts, quick links to registrations/analytics, recent events
System Health (Top Right): CPU, bandwidth, latency indicators with status, notifications bell, operator profile

Design Highlights:

  • ✓ 3-Column Layout: Event control (20%), main console (60%), quick access (20%)
  • ✓ Real-Time Metrics: Participant count, sentiment, engagement, compliance alerts
  • ✓ Professional Aesthetic: Navy blue background with teal accents, glassmorphism effects, pulsing live indicators
  • ✓ One-Click Access: All critical controls within reach, no context switching
  • ✓ Shadow Mode: Easily toggle monitoring-only mode for training or observation
  • ✓ Broadcast-Ready: Designed for professional event operations with compliance tracking

📅 Events & Bookings Management

Streamlined event scheduling and management interface for operators to view, create, edit, and manage all events from a unified calendar view.

Key Features:

Mini Calendar (Left): Month view with event indicators, quick filter buttons (Today, This Week, Upcoming, All Events)
Week View (Center): 7-day event cards showing event name, time, platform, participant count, status badges
Event Details Panel: Comprehensive event information with action buttons (Edit, View Participants, Download Report, Cancel)
Status Badges: LIVE (red pulsing), SCHEDULED (green), COMPLETED (gray) for quick visual identification
Quick Actions: Create New Event button, search functionality, event filtering
Event Statistics (Right): Total events this month, upcoming events, completed events, participant trends

Design Highlights:

  • ✓ Intuitive Calendar: Familiar month/week view for easy event navigation
  • ✓ Platform Icons: Visual indicators for Zoom, Teams, Webex, RTMP
  • ✓ Quick Event Launch: One-click access to launch any event
  • ✓ Comprehensive Details: All event information at a glance
  • ✓ Bulk Operations: Manage multiple events efficiently
  • ✓ Recording & Transcription Status: See which events are being recorded/transcribed

👥 Participants & Registrations Management

Comprehensive participant tracking interface for managing registrations, monitoring attendance, tracking engagement, and generating reports.

Key Features:

Quick Stats (Left): Registered count, attended count, no-show count, cancelled count at a glance
Attendance Rate Gauge: Visual representation of attendance percentage (e.g., 83% attended)
Engagement Trend: Sparkline showing attendance trend over time
Participant Table: Sortable table with name, company, email, registration date, status, sentiment score
Sentiment Indicators: Emoji-based sentiment feedback (😊 😐 😞) for quick engagement assessment
Bulk Actions: Send email to selected, mark as attended, export selected participants
Engagement Insights: Average sentiment, most active company, engagement rate, top questions
Export Functionality: Export to CSV for reporting and analysis

Design Highlights:

  • ✓ Data-Driven: Comprehensive participant metrics and engagement tracking
  • ✓ Quick Filtering: Filter by attendance status, registration status, sentiment
  • ✓ Bulk Operations: Efficiently manage large participant lists
  • ✓ Sentiment Analysis: Visual sentiment indicators for engagement assessment
  • ✓ Reporting Ready: Generate reports with attendance, engagement, and sentiment data
  • ✓ Search & Export: Find participants and export data for external use

📋 Operator Dashboard Design Overview

This redesigned operator dashboard consolidates all daily event management and operations tasks into a single, visually stunning, operator-centric interface.

Key Improvements from Previous Design:

✓ Operator-Centric Focus

Removed architecture and roadmap tabs. Focused entirely on day-to-day operational tasks that operators actually use.

✓ Visual Appeal

Modern, sleek aesthetic with navy blue backgrounds, teal accents, glassmorphism effects, and smooth animations. Professional broadcast control room feel.

✓ Integrated OCC

Operator Console (OCC) is now the central focus of the Live Console tab, with transcription, sentiment analysis, and audio/video controls prominently displayed.

✓ Webcasting & Video

Dedicated section showing RTMP stream status, bitrate, viewer count, video preview, and recording status. All in one place.

✓ Shadow Mode

Prominent toggle in the right sidebar for monitoring-only mode. Easily switch between control and observation modes.

✓ All Features in One Place

Event control, live monitoring, OCC, webcasting, video, shadow mode, compliance alerts, and quick links all accessible without context switching.

Color Scheme & Visual Language:

Primary Colors

Navy Blue (#0f172a): Main background color
Teal (#14b8a6): Accent color for interactive elements and highlights

Status Indicators

Green (#10b981): Active/Good status
Red (#ef4444): Live/Critical status
Yellow (#f59e0b): Warnings/Alerts

Visual Effects

Glassmorphism effects on cards, smooth gradients, pulsing indicators for live status, subtle shadows for depth, generous spacing and padding.
